== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Some Plugin'
3. Activate Some Plugin from your Plugins page. (You'll be greeted with a Welcome page.)

= From WordPress.org =

1. Download Some Plugin.
2. Upload the 'some-plugin' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate from your Plugins page. (You'll be greeted with a Welcome page.)
